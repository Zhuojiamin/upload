关于注册成为BLOG会员


    您需要 注册成为会员 ，如果你在论坛中有帐号，那你可以用论坛帐号登入.论坛和BLOG可以使用同一帐号.只有成为会员才可以使用BLOG的各种功能.注册时，您需要一个用户名、密码和一个有效的email地址。输入您的email地址不会给您带来‘垃圾邮件’，您可以在注册时选择隐藏email地址。

简要功能介绍：
    (1) 二级域名功能：可以使用个人二级域名可以直接访问博客个人主页
    (2) 可以增加和修改你的个人日志
    (3) 发表评论
    (4) 修改自己的BLOG风格
    (5) 创建自己的相册，上传照片等
    (6) 如果你有权限，你还可以创建一个朋友圈，邀请几个知心朋友，谈天说地，享受网络交流的便捷
    (7) 搜索您所需要的资料

关于登录BLOG


    (1) 如果您以前是该BLOG的会员，请点击 登录 进入登录界面，填写用户名和密码
    (2) 如果你在pw论坛上有会员帐号，你可以在论坛插件中心，博客插件里进入，如果你以前没用过这个帐号登入过BLOG，请先激活
    (3) Cookie 有效期介绍: 主要用于在您离开一会儿或者重新进入一个主题或者帖子的时候确认您的身份。当您登录时， 您将有一个‘cookie 有效期:’选项，您可以选择cookie保存的时间，这时系统将您的用户名和密码用cookie保存，储存在您的电脑。如果您是使用一台共用的电脑，比如在图书馆、网吧，或者其他不能完全信任的地方， 建议使用‘即时’选项，以免您的帐号被他人盗用。

关于日志管理


    添加日志：添加新的日志，可以自己增加日志内容等。
    日志管理：在你新的日志中，可以进行编辑，修改等操作。
    评论管理：你可以对前台“最新文章”中发表的评论进行管理
    专辑管理：这里你可以建立自己的专辑项目（分类），在你的个人博客里显示在最上面。

论坛帖子推送


    帖子推送：可以将论坛上的你发表的文章转到你的blog中来。

关于用户设置


    修改个人设置：这里你可以修改你的注册信息，个人资料
    用户自定义风格：你可以修改和增加风格。
    管理我的联结：可以添加和管理博客右下角的友情连接。
    管理我的公告：设置你的公告栏。

关于朋友圈管理


    我的朋友圈：这里你可以创建和编辑朋友圈，当然创建朋友圈要一定的权限。
    朋友圈会员管理：你可以对朋友圈里的会员进行审核的功能操作。
    我加入的朋友圈：在这里你可以退出你所在的朋友圈。
    朋友圈文章管理：你可以对朋友圈的文章进行管理。

上传文件管理


    将显示上传到blog上的文件信息，你可以对它进行各种操作如：删除

关于相册管理


    这里可以创建相册，以及对你的相册进行编辑,添加，删除图片等

关于Wind Code


    可以使用Wind Code 编辑您的帖子、签名，您可以设置帖子文字的字体、颜色等格式。Wind Code 代码使用简单。

关于创建朋友圈


    只有当会员有一定权限的时候才可以创建朋友圈。朋友圈是为了让一些特定的人聚集在一起，可以互相学习，交流，和结识新朋友。

关于BLOG上传附件问题


    上传附件首先要在BLOG后台管理里打开这个功能，上传的文件有特定的类型，后台可以添加(注意这里说的不是个人博客后台)，附件的大小也可以限制，只有符合要求的附件才可以上传。附件将显示在您所发布的信息中. 注意：放附件的文件要"777" 一般 附件保存方式使用"预设(全部存入同一目录)"

关于管理员如何添加风格


    找到template/default/skin/ 目录，复制或创建新文件夹命名为2， 如果你是自己制作新风格，那么就复制风格1文件夹内的所有文件到风格2文件夹里进行文件修改和编辑，如果是别人提供的模版，就按照提示放到 风格2目录里 （添加更多风格只需按照同样办法建立文件包并按顺序命名即可）进入管理后台 风格管理 创建新风格 完成后编辑该风格属性即可
